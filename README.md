Programa em Java utilizando Swagger e SpringBoot para integração de 2 APIs, uma de gerenciamento de dados de usuário, e outra de gerenciamento de dados de planilhas. As APIs trocam dados e requisições através da dependência do SpringBoot "Spring Cloud OpenFeign".

Agora as APIs também aceitam o envio de arquivos .xls, passando a atuar também com planilhas reais

Projeto feito pelos alunos Otto Basso e Leonardo Freire

Adição de uma estrutura de front end em html, css e JavaScript para os endpoints de requisições da API de planilhas,
contendo cadastro manual de usuários, planilhas, e requisições de comparação entre palnilhas, além de um receptor 
de arquivos .xls para extrair os dados direto para o banco.
